# FinalProject

This repository contains my notebook-based final project work.

## Setup

1. Create and activate the environment:
   - `uv venv .venv --clear`
   - PowerShell: `./.venv/Scripts/Activate.ps1`
2. Install dependencies:
   - `uv pip install --python .venv/Scripts/python.exe jupyter pandas numpy`

## Run

- Open `FinalProject.ipynb` in VS Code and select the `.venv` kernel.
